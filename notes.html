Css

Css helps you style your page. You can write css in html or in a seperate file.
To write it in the html file you must use what are called inline styles. They can target specific pieces of text and style them (colour,font,size). You would put style= "color: red; in between the p and the > of the opening p tag. That paragraph would now be in red.

To apply styling to an entire page, you must put the style tags in the head tag (which contains metadata).

Often times however, developers will seperate the css file from the html file. To link it, you must put the (link href= >) tag. Put the file name after "=". You have to then put the type of file (type="text/css") and the relationship between the two files (rel="stylesheet")

The css file

You can select elements of the html file to style. If you put p{blue for example}, everything in every paragraph will be blue, however it will be overruled by an inline style tag. You can also class parts of your text and name them. You can call this paragraph john, then in your css file you can say john is green. this paragraph will be green. You would refer to john as such: .john{}. In the swiggly brackets you put style tags. You can put as many classes as you want in an html element. You can also select elements with a class instead of a class alone. You simply put h1.max if you want to select h1s with that are in the max class.

Nesting

You can nest elements and tags and classes within eachother. You can select all h1s with classed max by putting (h1.max {}). However you can also select all items classed max that are h1s by putting ".max h1 {}". The combinations are endless!

The best way to easily style elements and tags is to create a class for every different colour, font, etc and name it of the same style (.green makes the text green). Then, all you do is put the class of whichever element/tag you want to be green as green.

The different properties of text styles

The different properties of text are "color", "font-family", "font-size", "font-allignment", "font-weight", "line-height", etc...

You can set the background image by using the background image property. The value (instead of a number) is the url between brackets and quotations.

Borders

Borders suround an element. To change this property you use "border:". The 3 properties of a border are width, style and colour. If you put "border: 3px solid coral;" your border will be 3px wide, solid and coral colour. You can also modify

Padding is the space between the text (content) and the border. The only property of padding is px (width). This affects all side of the text. You can specify which side you would like to modify by specifying. padiing-bottom, padding-top, padding-left, padding-right. You can put in 1 line, the four numbers in clockwise order.
padding: 6px 9px 5px 2px
If you leave out one side it will match the opposite side (top bottom, left right).

The same ideas apply to margin. Margin is the space around the border. Nothing can enter the margin (ex: the border of another element cannot interfere with the margin). The Top and bottom margins do not combine, left and right margins combine.

Min-width and Max-width, Min-height and Max-height control how big or small an element can become when stretched/shrunk on different screens.

Overflow is the property that decides what happens with content that that spills out of its borders. Three options: hidden, scroll, visible. They do exactly what they say (hide, add scroll bar, show out of box).
